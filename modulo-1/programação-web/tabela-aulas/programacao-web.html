Programação Web

Prof. Sueli Muniz

A disciplina de programação web ensina a criar páginas e serviços na internet, como blogs e lojas virtuais. Para isso, utiliza-se linguagens como HTML, CSS e JavaScript.

O que se aprende em programação web?

  • Construir páginas web com HTML
  • Formatar páginas web com CSS
  • Criar páginas web dinâmicas com JavaScript
  • Conhecer frameworks para desenvolvimento de aplicações web
  • Conhecer o modelo cliente-servidor, padrão MVC, arquitetura em camadas, protocolo http
  • Conhecer linguagens de programação para internet

Tecnologias utilizadas HTML (Linguagem de Marcação de HiperTexto), CSS (Casca de Estilos), JavaScript, PHP.

Objetivos da disciplina

  • Promover condições básicas para a programação web
  • Capacitar os estudantes em programação para web

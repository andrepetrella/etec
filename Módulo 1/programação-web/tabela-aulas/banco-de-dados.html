Banco de Dados

Prof. Jonathas

Um banco de dados é uma coleção organizada de dados, que pode ser armazenada eletronicamente em um computador. Ele permite que informações sejam coletadas, armazenadas, recuperadas e manipuladas de forma eficiente.

Tabela representativa

O que é

Ferramenta para organizar e armazenar informações

O que armazena

Dados sobre pessoas, produtos, pedidos, imagens, vídeos, arquivos, etc.

Como é organizado

Em tabelas, com linhas e colunas

Como é gerenciado

Por um sistema de gerenciamento de banco de dados (DBMS)

Para que serve

Para ajudar a tomar decisões, automatizar processos, e mais

Bancos de dados são usados em diversas áreas, como Lojas online, Hospitais, Empresas para gerenciar informações de clientes, Aplicações científicas e governamentais.

Alguns exemplos de bancos de dados são:

  • Oracle
  • DB2
  • MySQL
  • SQL Server
  • PostgreSQL
  • MongoDB
  • Redis

O projeto de um banco de dados deve considerar a modelagem conceitual e o projeto lógico, de modo a garantir boa performance e facilidade de manutenção.

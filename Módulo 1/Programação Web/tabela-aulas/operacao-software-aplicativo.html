Operação Software Aplicativo

Prof. Sueli Muniz

O software aplicativo é, basicamente, o programa utilizado para aplicações dentro do S.O, que não esteja ligado com o funcionamento do mesmo. Exemplos: Word, Excel, Paint, Bloco de notas, calculadora. Outros exemplos de software aplicativos são os Browsers, isto é, os navegadores utilizados para acesso à internet.

Tipos de software:

  • Software aplicativo
  • Software de programação
  • Software de sistema

Classificação:

Aplicações de softwares são divididas em duas categorias: aplicações horizontais e aplicações verticais. Aplicações horizontais são as mais populares em departamentos e empresas. Aplicações verticais são produtos que atendem a um determinado nicho, para um tipo especifico de negócio ou divisão de uma companhia.
